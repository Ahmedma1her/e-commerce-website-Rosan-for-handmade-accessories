# Rosan – Front-End Template for Handmade Products E-Commerce

## Project Overview
**Rosan** is a **Front-End Template** for an e-commerce website showcasing handmade products like accessories, necklaces, bracelets, and knitted items.  

This template focuses on **visual design, layout, and user experience**, with multiple sections including: Home, About, Products, Services, Blog, and Contact.  

> ⚠️ **Note:** This is a **Front-End template only**. It does **not include a Back-End**, database, or real product links. It is designed for demonstration purposes and easy customization.

---

## Technologies Used
- **HTML5 & CSS3** for layout and styling  
- **JavaScript** for interactive elements  
- **Responsive Design** to support desktop and mobile devices  
- **Fonts & Icons:** Google Fonts (Pacifico) and Font Awesome  

---

## Features
- **Clean & Attractive Layout:** Highlights handmade products in a visually appealing way.  
- **Responsive Design:** Works smoothly on desktops, tablets, and smartphones.  
- **Multiple Sections:** Home, About, Products, Services, Blog, Contact.  
- **Interactive Effects:** Hover effects on cards and buttons for better user experience.  
- **Easy to Customize:** Change images, text, and colors to fit your brand.  

---

## Usage
1. Clone or download this repository.  
2. Open `index.html` in any web browser to view the template.  
3. Customize images, text, and styles as needed.
